# 生日助手　birthday-assist

生日助手的设计目的是辅助用户记住别人的生日来增进彼此的社交关系．
(It's a web application designed to assist users to memorize birthdays for a better social relationship.)

##　采用的技术或工具
- 语言：html5, css, js
- 库或框架：bootstrap, jquery
- 编辑器: sublime text 3
- 版本管理: git
